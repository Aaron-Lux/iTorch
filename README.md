iTorch
======

iPython Kernel for Torch

###Installation
luarocks install itorch

OR

luarocks make https://raw.githubusercontent.com/soumith/iTorch/master/itorch-scm-1.rockspec

### Usage
At commandline, type:
itorch

OR 

itorch notebook

OR

itorch qtconsole

(basically whatever ipython mode you want)

Win at life!
